## 1. Что такое компетенции?

**Компетенции** — это **знания, навыки и личные качества**, необходимые человеку для успешного выполнения задач.

Выделяют четыре области (треугольник компетенций):

| Компетенция | Что означает |
|-------------|--------------|
| **Профессиональная компетенция** | Знания по специальности, опыт, образование |
| **Социальная компетенция** | Умение работать в команде, общаться, решать конфликты |
| **Личностная компетенция** | Самоорганизация, надёжность, мотивация |
| **Методическая компетенция** | Владение методами управления проектами, приёмами решения проблем |

---

## 2. Типичные задачи руководителя проекта

| Задача | Описание |
|--------|----------|
| **Планирование** | Составление структуры проекта, плана по срокам, бюджету, ресурсам |
| **Управление** | Контроль сроков, бюджета, качества |
| **Руководство** | Мотивация команды, распределение задач |
| **Коммуникация** | Отчёты заказчику, проведение совещаний |
| **Управление рисками** | Выявление, оценка и реагирование на риски |
| **Приёмка** | Презентация результатов, организация приёмки заказчиком |

---

## 3. Компетенции, которыми должен обладать руководитель проекта

| Область компетенций | Примеры |
|---------------------|---------|
| **Профессиональная** | Знание методов управления проектами (водопад, V‑модель, гибкие методы), понимание темы проекта |
| **Социальная** | Умение общаться, вести команду, разрешать конфликты |
| **Личностная** | Готовность принимать решения, стрессоустойчивость, инициативность |
| **Методическая** | Навыки модерации, презентации, ведения переговоров |

---

## 4. Типичные задачи участника команды

| Задача | Описание |
|--------|----------|
| **Реализация** | Выполнение конкретных задач (разработка, тестирование, документирование) |
| **Отчётность** | Информирование руководителя о прогрессе |
| **Координация** | Совместная работа с другими участниками команды |
| **Обеспечение качества** | Самопроверка своей работы, документирование |
| **Соблюдение сроков** | Своевременное выполнение рабочих пакетов |

---

## 5. Компетенции участника команды

| Область компетенций | Примеры |
|---------------------|---------|
| **Профессиональная** | Знание технологий (Java, SQL, Delphi), понимание предметной области |
| **Социальная** | Умение работать в команде, готовность помогать, конструктивная обратная связь |
| **Личностная** | Надёжность, внимательность, готовность учиться |
| **Методическая** | Навыки решения проблем, работа с системами учёта задач, тестирование и отладка |

## 7. Главная мысль для экзамена

> **Руководитель проекта** — планирует, управляет, ведёт команду и общается.  
> Ему нужны **методические, социальные и лидерские компетенции**.

> **Участники команды** — выполняют задачи, сотрудничают и соблюдают сроки.  
> Им нужны **профессиональные знания и умение работать в команде.** ✅
